University Courses Offered

1. Web Development

This course covers the fundamentals of web development, including HTML, CSS, and JavaScript. Students will learn how to create responsive and interactive websites.

Topics Covered:

  • HTML Basics
  • CSS Styling
  • JavaScript Fundamentals
  • Responsive Design
  • Web Accessibility

2. Data Science

This course introduces students to data analysis, visualization, and machine learning using Python and R. Students will work on real-world datasets to gain practical experience.

Topics Covered:

  • Data Manipulation with Pandas
  • Data Visualization with Matplotlib and Seaborn
  • Statistical Analysis
  • Machine Learning Algorithms
  • Data Ethics

3. Graphic Design

This course focuses on the principles of graphic design, including typography, color theory, and layout. Students will use Adobe Creative Suite to create visual content.

Topics Covered:

  • Design Principles
  • Adobe Photoshop Basics
  • Adobe Illustrator Basics
  • Branding and Identity
  • Print vs. Digital Design

4. Digital Marketing

This course explores the strategies and tools used in digital marketing, including SEO, social media marketing, and email marketing. Students will learn how to create effective marketing campaigns.

Topics Covered:

  • Search Engine Optimization (SEO)
  • Content Marketing
  • Social Media Strategies
  • Email Marketing Techniques
  • Analytics and Metrics

5. Introduction to Psychology

This course provides an overview of the fundamental concepts in psychology, including human behavior, cognition, and emotion. Students will explore various psychological theories and applications.

Topics Covered:

  • History of Psychology
  • Research Methods
  • Developmental Psychology
  • Cognitive Psychology
  • Social Psychology
