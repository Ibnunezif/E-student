Course Content

Lesson 1: Introduction to HTML

HTML (Hypertext Markup Language) is the standard language for creating web pages. HTML describes the structure of a web page using markup.

Topics:

  • HTML Elements
  • HTML Attributes
  • HTML Tags

HTML elements are the building blocks of HTML pages. An HTML element is defined by a start tag, some content, and an end tag.

Lesson 2: Introduction to CSS

CSS (Cascading Style Sheets) is a language used to describe the style of an HTML document. CSS describes how HTML elements should be displayed.

Topics:

  • CSS Selectors
  • CSS Properties
  • Styling Text

CSS selectors are used to "find" (or select) the HTML elements you want to style. We can select elements based on their name, id, class, attribute, and more.

Lesson 3: Building Your First Web Page

Combining your HTML and CSS knowledge, you'll build a simple web page from scratch.

Topics:

  • Creating HTML Structure
  • Adding CSS Styles
  • Enhancing with Basic JavaScript

In this lesson, we will create a simple HTML page with a header, a navigation menu, and some content. We'll style it using CSS and add basic interactivity with JavaScript.
